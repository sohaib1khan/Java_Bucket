-----------------------------------------

PROGRAM BankingApp:

        1. Print welcome message

           - Account number: "ACC001"
           - Owner: "Alice"
           - Initial deposit: $1000
        
        3. Display account info

           - Print result

           - Print result
        
        7. Create second account:
           - Account number: "ACC002"
           - Owner: "Bob"
           - Initial deposit: $500
        
        8. Display both accounts
